Sobre o Projeto

Ferramenta automatizada para identificacao, classificacao e anonimizacao de dados pessoais (PII) em conformidade com a Lei Geral de Protecao de Dados (LGPD). O sistema detecta dados sensiveis e aplica tecnicas de anonimizacao para protecao da privacidade.

Referencias LGPD

  • Art. 5, II: Definicao de dados anonimizados
  • Art. 12: Anonimizacao como alternativa ao consentimento
  • Art. 6, III: Principio da necessidade
  • Art. 46: Medidas de seguranca

Metodos de Anonimizacao

Metodo Descricao Exemplo Reversivel?
mask Substitui caracteres por * 123.456.789-00 → 123.***.***-00 Nao
hash Hash SHA-256 com salt joao@email.com → a1b2c3d4e5f6 Nao
pseudonymize Substitui por dado falso Joao Silva → Maria Santos Nao
generalize Reduz precisao (faixas) R$ 5.432,00 → Faixa 3k-6k Nao
suppress Remove completamente (11) 99999-9999 → [SUPRIMIDO] Nao
tokenize Substitui por token unico email@test.com → EMAIL_00001 Sim
noise Adiciona variacao aleatoria R$ 5.000,00 → R$ 4.872,34 Nao

Classificacao de Risco LGPD

  • CRITICO: CPF, CNPJ, RG - Identificadores unicos
  • ALTO: Nome, E-mail, Salario - Dados sensiveis
  • MEDIO: Telefone, Endereco, Data Nascimento
  • BAIXO: CEP, Cidade, Estado
